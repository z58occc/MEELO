# 🚀 專案簡介
本專案由六角學院切版班小組與 AAPD 設計學員合作完成，
以前端工程師與設計師的角色協力實作 MEELO 交友軟體，
並部署到 GitHub Pages 展示。\
專案目標是打造一個以內在共鳴連結，並能安心互動的社交場所，
透過角色引導與溫柔的互動節奏，讓使用者自然建立更深刻且真實的連結。
